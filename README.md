# Shoot the Duck (Python / Pygame)

A simple arcade shooter. Move the crosshair with your mouse and click ducks.

## Requirements
- Python 3.10+ recommended

## Install
```bash
pip install -r requirements.txt



Run
python shoot_the_duck.py
Controls

Left click: shoot

R: reload

P: pause

Enter: restart after game over

Esc: quit


3. Commit
